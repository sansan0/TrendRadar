# Changelog v5.0.2

**Release Date:** 2026-01-14

## Summary

Documentation reorganization and naming convention improvements for better
consistency.

## Changes

### Added

-   New changelog structure with version-based files
-   `docs/changelog/` directory for version-specific changelogs
-   `docs/CHANGELOG.md` as index file
-   **HTML AI Analysis Extension** (`extensions/html_ai_analysis/`)
    -   Adds AI-powered analysis section to HTML reports
    -   Injects AI analysis results at top of rendered HTML
    -   Customizable section title, colors, and report type
    -   Uses same AI analyzer as notifications
    -   Config: `config/extensions/html_ai_analysis.yaml`
    -   Docs: `docs/extensions/html-ai-analysis.md`

### Changed

-   Renamed `docs/extensions.md` to `docs/README-EXTENSIONS.md` for naming
    consistency
-   Renamed `docs/nas/DEPLOYMENT.md` to `docs/SYNOLOGY-NAS-DEPLOYMENT.md` for
    clarity
-   Updated `CHANGELOG.md` to link to version-specific files in `changelog/`
    folder
-   Moved version-specific changelog from `docs/5.0.1.md` to
    `docs/changelog/5.0.1.md`

### Documentation Structure

```
docs/
├── CHANGELOG.md                    # Index (version overview)
├── README-EXTENSIONS.md            # Extension system documentation
├── README-EN.md                    # Main README (English)
├── README-MCP-FAQ.md               # MCP FAQ
├── README-MCP-FAQ-EN.md            # MCP FAQ (English)
├── README-Cherry-Studio.md         # Cherry Studio integration
├── SYNOLOGY-NAS-DEPLOYMENT.md      # Synology NAS deployment guide
└── changelog/
    ├── 5.0.1.md                    # v5.0.1 detailed changelog
    └── 5.0.2.md                    # This file
```
