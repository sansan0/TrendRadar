# TrendRadar MCP 도구 사용 문답

> AI 질문 가이드 - 대화를 통해 뉴스 핫토픽 분석 도구를 사용하는 방법

## ⚙️ 기본 설정 설명(중요!)

기본적으로 다음 최적화 전략을 사용하며, 주로 AI 토큰 소비를 절약하기 위함입니다:

| 기본 설정      | 설명                                    | 조정 방법                              |
| -------------- | --------------------------------------- | ------------------------------------- |
| **반환 조건 수**   | 기본값: 50건 뉴스 반환                      | 대화 중 "상위 10건 반환" 또는 "100건 주기" 라고 말하기 |
| **시간 범위**   | 기본값: 오늘 데이터 조회                      | "어제 조회", "최근 일주일" 또는 "1월 1일~7일" 이라고 말하기 |
| **URL 링크**   | 기본값: 링크 미반환(약 160 토큰/건 절약)  | "링크 필요" 또는 "URL 포함" 이라고 말하기              |
| **키워드 목록** | 기본값: frequency_words.txt로 뉴스 필터링 안함 | "트렌드 토픽" 도구 호출 시에만 사용        |

**⚠️ 중요:** AI 모델 선택이 도구 호출 효과에 직접 영향을 미칩니다. AI가 똑똑할수록 호출이 정확합니다. 위의 제한을 해제할 때, 예를 들어 오늘 조회에서 일주일 조회로 확대할 경우, 먼저 로컬에 일주일 데이터가 있어야 하고, 다음으로 토큰 소비량이 2배 이상 증가할 수 있습니다(왜 "수 있습니다"라고 표현하나면, '사과'의 최근 일주일 열도 트렌드를 분석할 때 일주일 동안 사과 관련 뉴스가 많지 않으면 토큰 소비량이 오히려 매우 적을 수 있기 때문입니다)

**💡 팁:** "최근 7일간"이라고 말하면 AI가 자동으로 해당 날짜 범위(예: 2025-10-18 ~ 2025-10-25)를 계산하고 도구에 전달합니다.

## 기초 쿼리

### Q1: 최신 뉴스를 어떻게 확인하나요?

**이렇게 물어볼 수 있습니다:**

- "최신 뉴스를 보여줄래"
- "오늘의 핫토픽 뉴스를 조회해줄래"
- "지혜(知乎)와 웨이보의 최신 10건 뉴스를 가져와"
- "최신 뉴스를 보고 싶은데 링크도 포함해줄래"

**호출되는 도구:** `get_latest_news`

**도구 반환 동작:**

- MCP 도구가 모든 플랫폼의 최신 50건 뉴스를 AI에 반환함
- URL 링크는 포함하지 않음 (token 절약)

**AI 표시 동작(중요):**

- ⚠️ **AI는 보통 자동으로 요약하여** 일부 뉴스만 표시함 (예: TOP 10-20건)
- ✅ 전체 50건을 보고 싶으면 명확하게 요청해야 함: "모든 뉴스를 보여줄래" 또는 "모든 50건을 완전히 나열해줄래"
- 💡 이것은 AI 모델의 자연스러운 동작이지, 도구의 제한이 아닙니다

**조정할 수 있습니다:**

- 플랫폼 지정: "지혜(知乎)의 뉴스만 보여줄래"
- 개수 조정: "상위 20건을 반환해줄래"
- 링크 포함: "링크가 필요해"
- **전체 표시 요청**: "전부 보여줄래, 요약하지 말고"

**호출되는 도구:** `get_news_by_date`

**지원되는 날짜 형식:**

- 상대 날짜: 오늘, 어제, 그제, 3일 전
- 요일: 지난 주 월요일, 이번 주 수요일, last monday
- 절대 날짜: 2025-10-10, 10월 10일

**도구 반환 동작:**

- 날짜를 지정하지 않으면 자동으로 오늘 조회 (token 절약)
- MCP 도구가 모든 플랫폼의 50건 뉴스를 AI에 반환함
- URL 링크는 포함하지 않음

**AI 표시 동작(중요):**

- ⚠️ **AI는 보통 자동으로 요약하여** 일부 뉴스만 표시함 (예: TOP 10-20건)
- ✅ 전체를 보고 싶으면 명확하게 요청해야 함: "모든 뉴스를 보여줄래, 요약하지 말고"

### Q3: 내가 관심있는 토픽의 빈도 통계를 어떻게 확인하나요?

**이렇게 물어볼 수 있습니다:**

- "내가 관심있는 단어들이 오늘 몇 번 나타났어"
- "내 관심 단어 목록에서 어떤 단어가 가장 인기있어"
- "frequency_words.txt의 관심 단어 빈도를 집계해줄래"

**호출되는 도구:** `get_trending_topics`

**중요 사항:**

- 이 도구는 **자동으로 뉴스 핫토픽을 추출하는 것이 아님**
- 대신 `config/frequency_words.txt`에서 설정한 **개인 관심 단어**를 통계함
- 이것은 **커스터마이징 가능한** 목록으로, 관심에 맞게 단어를 추가할 수 있습니다

## 검색 검색

### Q4: 특정 키워드가 포함된 뉴스를 어떻게 검색하나요?

**이렇게 물어볼 수 있습니다:**

- "'인공지능'이 포함된 뉴스를 검색해줄래"
- "'테슬라 가격 인하'에 대한 보도를 찾아줄래"
- "머스크 관련 뉴스를 검색해줄래, 상위 20건 반환"
- "최근 7일간 'iPhone 16'에 관한 뉴스를 찾아줄래"
- "2025년 1월 1일부터 7일까지 '테슬라' 관련 뉴스를 찾아줄래"
- "'iPhone 16 발표' 뉴스의 링크를 찾아줄래"

**호출되는 도구:** `search_news`

**도구 반환 동작:**

- 키워드 패턴으로 검색
- 기본값으로 오늘 데이터 검색
- AI가 자동으로 "최근 7일", "지난주" 같은 상대 시간을 구체적인 날짜 범위로 변환
- MCP 도구가 최대 50건의 결과를 AI에 반환
- URL 링크는 포함하지 않음

**AI 표시 동작(중요):**

- ⚠️ **AI는 보통 자동으로 요약하여** 검색 결과의 일부만 표시
- ✅ 전체를 보고 싶으면 명확하게 요청해야 함: "모든 검색 결과를 보여줄래"

**조정할 수 있습니다:**

- 시간 범위 지정:
  - 상대 방식: "최근 일주일의 것들을 검색해줄래" (AI가 자동으로 날짜 계산)
  - 절대 날짜: "2025년 1월 1일부터 7일까지 검색해줄래"
- 플랫폼 지정: "지혜(知乎)에서만 검색해줄래"
- 정렬 조정: "가중치 순서로 정렬해줄래"
- 링크 포함: "링크가 필요해"

**대화 예시:**

```
사용자: 최근 7일간 "인공지능 돌파"에 관한 뉴스를 검색해줄래
AI: (자동 계산: date_range={"start": "2025-10-18", "end": "2025-10-25"})

사용자: 2025년 1월 "테슬라" 보도를 찾아줄래
AI: (date_range={"start": "2025-01-01", "end": "2025-01-31"})
```

## 트렌드 분석

### Q6: 토픽의 핫도 트렌드를 어떻게 분석하나요?

**이렇게 물어볼 수 있습니다:**

- "'인공지능'의 최근 일주일 핫도 트렌드를 분석해줄래"
- "'테슬라' 토픽이 순간적인 유행일까 아니면 지속적인 핫토픽일까 봐줄래"
- "오늘 갑자기 핫해진 토픽이 뭐가 있어"
- "앞으로 나올 가능성 있는 핫토픽을 예측해줄래"
- "'비트코인'의 2024년 12월 생명 주기를 분석해줄래"

**호출되는 도구:** `analyze_topic_trend`

**도구 반환 동작:**

- 여러 분석 모드 지원: 핫도 트렌드, 생명 주기, 이상 탐지, 예측
- AI가 자동으로 "최근 일주일" 같은 상대 시간을 구체적인 날짜 범위로 변환
- 기본값으로 최근 7일 데이터 분석
- 일 단위 통계

**AI 표시 동작:**

- 보통 트렌드 분석 결과와 차트를 표시
- AI가 주요 발견을 요약할 수 있음

**대화 예시:**

```
사용자: '인공지능'의 최근 일주일 생명 주기를 분석해줄래
AI: (자동 계산: date_range={"start": "2025-10-18", "end": "2025-10-25"})

사용자: '비트코인'이 2024년 12월에 순간 유행일까 아니면 계속 핫할까
AI: (date_range={"start": "2024-12-01", "end": "2024-12-31"})
```

**두 가지 모드:**

| 모드           | 용도                 | 예시                 |

- ✅ 전체를 보고 싶으면 명확하게 요청해야 함: "크롤링한 모든 뉴스를 보여줄래"

**조정할 수 있습니다:**

- 플랫폼 지정: "지혜(知乎)에서만 크롤링해줄래"
- 데이터 저장: "저장도 함께 해줄래" 또는 "로컬에 저장해줄래"
- 링크 포함: "링크가 필요해"

## 💡 사용 팁

### 1. AI가 자동 요약이 아니라 전체 데이터를 표시하도록 하려면?

**배경**: AI가 자동으로 데이터를 요약하여 일부만 표시하는 경우가 있습니다. 도구가 완전한 50건의 데이터를 반환했음에도 그렇습니다.

**AI가 계속 요약하는 경우, 다음을 시도할 수 있습니다**:

- **방법 1 - 명확히 요청**: "모든 뉴스를 보여줄래, 요약하지 말고"
- **방법 2 - 개수 지정**: "50건의 모든 뉴스를 보여줄래"
- **방법 3 - 행동 의문**: "왜 15건만 보여줬어? 전부 봐야 돼"
- **방법 4 - 사전 설명**: "오늘 뉴스를 조회해줄래, 모든 결과를 완전히 보여줄래"

**참고**: AI는 여전히 맥락에 따라 표시 방식을 조정할 수 있습니다.


### 2. 여러 도구를 어떻게 함께 사용하나요?

**예시: 특정 토픽을 깊이 있게 분석하기**

1. 먼저 검색: "'인공지능' 관련 뉴스를 검색해줄래"
2. 그 다음 트렌드 분석: "'인공지능'의 핫도 트렌드를 분석해줄래"
3. 마지막으로 감정 분석: "'인공지능' 뉴스의 감정 경향을 분석해줄래"

**예시: 특정 사건 추적하기**

1. 최신 확인: "오늘 'iPhone' 관련 뉴스를 조회해줄래"
2. 역사 찾기: "지난주 'iPhone'과 관련된 역사 뉴스를 찾아줄래"
3. 유사 보도 찾기: "'iPhone 발표회'와 유사한 뉴스를 찾아줄래"
